nginx安装rtmp模块

  • 作者：Mike
  • 时间：2021-01-09 13:18:00
  • 99次访问

[TOC]

1、下载资源

  • 一、从 http://nginx.org/en/download.html 下载Nginx资源

  • 二、下载rtmp补丁

    wget https://github.com/arut/nginx-rtmp-module/archive/master.zip

2、编译

  • 一、需安装gcc和openssl以及zip解压
yum -y install gcc  
yum -y install gcc-c++  

yum  -y install openssl openssl-devel

yum install -y unzip zip
  • 二、解压资源
tar -zxvf nginx-1.18.0.tar.gz
unzip master.zip
  • 三、进入Nginx
cd nginx-1.18.0
mkdir build
./configure --prefix=/usr/local/mynginx/nginx-1.18.0/build --add-module=/usr/local/mynginx/nginx-rtmp-module-master

看到这个就成功了。

make -f objs/Makefile
make install

这时，build目录中的就是编译完成的Nginx程序。
