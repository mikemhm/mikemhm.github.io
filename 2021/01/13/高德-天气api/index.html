接口版本：

版本号 制定人 制定日期 修订日期
2.1.0 秦亮 2017-03-20 xxxx-xx-xx

请求URL:

  • https://restapi.amap.com/v3/weather/weatherInfo

请求方式：

  • GET

请求头：

参数名 是否必须 类型 说明

请求参数:

参数名 是否必须 类型 说明
key 是 string 秘钥 4d6ab733dcfed0e82806b9a97ff602ff
city 是 string 城市编码
extensions 否 string 可选值：base/all；base:返回实况天气；all:返回预报天气
output 否 string 返回格式；可选值：JSON,XML
